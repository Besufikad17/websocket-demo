# usage

- Clonning the repo
```bash
   git clone https://github.com/Besufikad17/websocket-demo
```

- Installing packages
```bash
   cd websocket-demo && npm install
```

- Setting up the database
```bash
   npx prisma migrate dev
```

- Running the server
```bash
   npm run dev
```
